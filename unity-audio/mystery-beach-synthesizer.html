Mystery Beach Synthesizer - A Collaborative RNBO to Unity Plugin Test

I created Mystery Beach Synthesizer to test the collaborative waters for RNBO integration within Unity. After several successful RNBO-to-Unity experiments, I reached out to my friend Connor, who is creating a video game for a research project at De Montfort University.

For a first test, he asked me to create a Unity package for a basic synthesizer, complete with oscillator choice, oscillator frequency control, a low-pass filter, and a feedback delay. Using Codex, I built a mysterious beach environment with abstract 3D control mechanisms, allowing first-person exploration with mouse and keyboard.

Each object in the scene controls a different synth parameter, turning the RNBO patch into a small playable sound toy. This test was also my first proper exploration of using GitHub repositories, enabling Connor and me to share the full Unity project and begin working in the same environment collaboratively.
