HARSH

A harsh noise VST3 made in RNBO, built around genexpr~ synthesis and a custom JUCE interface.

Description

HARSH was not the first VST3 I made using Cycling '74's RNBO to JUCE/CMake workflow, but it was a significant milestone. Previous builds used the basic JUCE sliders generated by the template; HARSH was the first one where I built a custom UI around the instrument.

It was also the first time I used genexpr~ inside an RNBO-based VST3. The sound engine is a self-contained harsh noise generator rather than an audio effect. The core noise source is produced by a drifting oscillator with feedback, low-pass tone shaping, and tanh saturation.

That signal is then passed into a second distortion stage with low/sub emphasis, resonant bump filtering, dry/wet mixing, and another layer of drive.

Fourteen parameters were exposed from RNBO, covering noise modulation, drift, feedback, tone shaping, output level, and the second distortion stage.

The UI is probably a bit harsh in itself, but I was genuinely chuffed with it because it felt like mine. I made a full custom interface with a black/red/white burst background, distressed HARSH logo, custom vertical faders for the noise section, and rotary controls for the distortion section.

After the plugin received some attention on Reddit's /r/noise, I decided it was worth polishing up properly and releasing for free.
