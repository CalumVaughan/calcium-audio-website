Cursed Johnny

An experimental audiovisual web piece where algorithmic remixing mutates a scanned 3D head in real time.

Summon Cursed Johnny

Description

Cursed Johnny is an experimental audiovisual web piece built from an ongoing series of algorithmic Max/MSP/Jitter collaborations with musician Johnny McJohnston.

Using a collection of 120 BPM source material provided by Johnny, I built a BPM-synchronised remix engine in RNBO designed to continuously re-sequence and mutate the incoming audio material in real time.

Alongside the audio, Johnny provided a 3D scan of his head, which I processed in Blender to isolate a set of deformable facial parameters, including nose length, mouth opening, and eye scale. These morph targets were then integrated into a browser-based WebGL environment using Three.js and directly mapped to the RNBO audio engine.

The result is Cursed Johnny: a reactive audiovisual entity whose facial structure continuously mutates in response to algorithmic audio transformations, blurring the boundary between musical remixing and character deformation.

This project continues my broader exploration into procedural audiovisual systems, where sound and form evolve as part of a shared generative process.
